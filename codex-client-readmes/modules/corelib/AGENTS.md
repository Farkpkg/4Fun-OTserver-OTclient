# Guia do Corelib (Lua base do client)

## Escopo

Este guia cobre utilitarios base em `modules/corelib`. Ele complementa o C++ do framework em `src/framework/AGENTS.md`.

## Onde editar

- Funcoes base e helpers: `modules/corelib`.
- APIs de engine e UI: C++ em `src/framework`.
 - UI utilitaria: `modules/corelib/ui/uimovabletabbar.lua` (tabs do console).
