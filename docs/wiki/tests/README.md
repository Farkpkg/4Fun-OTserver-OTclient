# Testes

## Visão geral
O repositório inclui estruturas de teste para servidor e cliente, organizadas em pastas específicas.

## Localização no repositório
- **Servidor**: `crystalserver/tests/`
- **Cliente**: `otclient/tests/`

## Estrutura interna (alto nível)
- Testes do servidor cobrem componentes de gameplay e infraestrutura.
- Testes do cliente cobrem módulos e comportamento do framework.

## Integrações
- **Servidor**: integra com CMake e dependências de teste.
- **Cliente**: integra com o build do OTClient e bibliotecas auxiliares.
