# Project Rules (Obrigatório)

## Regras Gerais
- Todo dev DEVE usar o Codex
- Todo dev DEVE ler a pasta /ai
- Pull Requests sem padrão serão recusados

---

## Desenvolvimento

- Não criar atalhos inseguros
- Não duplicar código

- Código limpo e reutilizável

---

## Protocolos

- Opcode >= 200
- Nunca alterar protocolo sem atualizar PROTOCOLS.md

## Code Review

- Performance
- Organização
